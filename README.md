# social
Node social network

MongoDB (3.2.11)
Node (7.2)

### commands to start the project after cloning

1) 1st cmd-prompt: cd => MongoDB\PATH\bin
	    => \bin>mongod 

2) 2nd cmd-prompt: cd => MongoDB\PATH\bin
      => \bin>mongo
              => >use node_connect
 		          
3) 3rd cmd-prompt: cd => \PATH_toProject>node server.js

4) Browser: => localhost:3000
